# Лексическая структура яызка

## Ключевые слова

| Слово    | Описание                           |
| -------- | ---------------------------------- |
| input    | ввод данных                        |
| print    | вывод данных                       |
| const    | объявление неизменяемой переменной |
| let      | объявление изменяемой переменной   |
| int      | тип целое число                    |
| float    | тип число с плавающей точкой       |
| string   | тип строка                         |
| bool     | логический тип                     |
| true     | литерал истины                     |
| false    | литерал лжи                        |
| if       | условный оператор                  |
| else     | альтернативная ветвь               |
| func     | объявление функции                 |
| return   | возврат значения из функции        |
| while    | цикл с предусловием                |
| break    | немедленный выход из цикла         |
| continue | переход к следующей итерации цикла |
| void     | тип отсутствие значения            |

Все ключевые слова регистрозависимые, т.е. `print` и `PRINT` не одно и то же

## Идентификаторы

* Являются последовательностью английских букв, арабских цифр и подчеркивания `[a-zA-Z0-9_]`
* Начинаются с буквы `[a-zA-Z]` или символа подчеркивания `_`, цифра не может быть первым символом
* Регистрозависимые
* Не могут совпадать с ключевыми словами

## Литералы

### Целые числа

* Являются последовательностью арабских цифр `[0-9]+`
* Всегда записываются в десятичной системе
* Нет литералов отрицательных чисел — вместо них используется оператор `-` (унарный минус)

### Вещественные числа

* Как целые числа используют арабские цифры и состоят из целой части, разделителя в виде точки и дробной части: `[0-9]+\.[0-9]+`
* Знак минуса `-` так же, как в целых числах не входит в литерал

### Строковые литералы

* Являются последовательностью любых символов, заключённая в двойные кавычки `"..."`.
* Внутри строки не могут содержаться неэкранированные двойные кавычки

### Логические литералы
* `true` – истина
* `false` – ложь

## Операторы

| Оператор | Описание                      |
| -------- | ----------------------------- |
| `+`      | сложение / конкатенация строк |
| `-`      | вычитание или унарный минус   |
| `*`      | умножение                     |
| `/`      | деление                       |
| `**`     | возведение в степень          |
| `=`      | присваивание                  |
| `&&`     | логическое И                  |
| `||`     | логическое ИЛИ                |
| `!`      | логическое НЕ                 |
| `<`      | меньше                        |
| `>`      | больше                        |
| `<=`     | меньше или равно              |
| `>=`     | больше или равно              |
| `==`     | равно                         |
| `!=`     | не равно                      |

## Прочие лексемы

* `;` - конец инструкции
* `,` - разделитель элементов в `print` и аргументов функций
* `:` - разделитель между идентификатором и типом
* `()` – группировка выражений, списки параметров
* `{}` – блоки кода

## Пробельные символы

Пробел ` `, табуляция `\t`, перевод строки `\n`, `\r\n` игнорируются, служат разделителями токенов на уровне лексера

## Комментарии

Поддерживаются два вида комментариев: 

* Однострочные комментарии: начинаются с  `///` и прололжаются до конца строки
* Многострочные комментарии: заключаются в `/*` и `*/`

Все комментарии игнорируются лексером
